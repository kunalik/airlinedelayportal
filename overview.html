Problem Statement

Delays in U.S. airlines cause significant inconvenience for passengers and financial impact for airlines. Understanding the patterns and causes of delays is essential to improve operational efficiency.

Tools & Technologies

  • Amazon Web Services (AWS)
  • Apache
  • SQL
  • Python

Approach

  1. Collected airline data for analysis.
  2. Cleaned and pre-processed datasets.
  3. Queried and analyzed the data.
  4. Visualized findings to highlight key insights.
